# Item Locator

アイテムの位置を簡単に管理できるPWAアプリケーションです。

## 機能

- アイテムの位置情報の管理
- オフライン対応
- モバイルフレンドリーなUI
- PWA対応（ホーム画面に追加可能）

## 技術スタック

- HTML5
- CSS3
- JavaScript
- Service Worker
- PWA

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/item-locator.git
```

2. ローカルサーバーで実行
```bash
# 例：Pythonの簡易サーバーを使用する場合
python -m http.server 8000
```

3. ブラウザでアクセス
```
http://localhost:8000
```
